# cascader-plus

## CascaderPlus介绍

扩展 ant-design 级联组件 cascader，实现级联选择的多选功能，并且支持本地及远程搜索。

## 功能

- [x] 多选功能

- [x] 多选搜索
- [x] 多选远程搜索

## 快速开始

安装依赖：

```bash
$ npm i
```

启动开发服务,

```bash
$ npm start
```

构建文档,

```bash
$ npm run docs:build
```
